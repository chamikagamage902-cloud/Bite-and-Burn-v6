# Bite & Burn 🥗🔥

**Bite & Burn** is an AI-powered health and fitness platform designed to provide personalized food and workout plans. The application uses AI to analyze user data, country-specific traditions, and health symptoms to offer a truly tailored wellness experience.

## 🚀 Vision
> "Eat Smarter, Work Harder and Live Healthier."

## ✨ Key Features

### 🍏 Personalized Food Planning
- **Country-Based Diets**: Traditional meal plans (e.g., Rice & Curry for Sri Lanka, Biryani for India, Sushi for Japan).
- **Nutrient Tracking**: Real-time analysis of carbs, proteins, and calorie intake.
- **Dietary Adaptive**: Adjusts for vegetarians, vegans, allergies, and medical conditions.

### 💪 Dynamic Workout Plans
- **AI-Driven Routines**: Exercises generated based on weight, height, and fitness goals.
- **Interactive Timer**: Per-exercise timers to guide your sessions.
- **Progress Tracking**: Visual completion status for all routines.

### 📅 Daily Routine Management
- **Smart Scheduling**: Differentiates between weekdays, weekends, and special days.
- **Percentage Progress**: Visual feedback on your daily task completion.
- **Adaptive UI**: Page glow changes from green to red as task intensity increases.

### 🧠 AI Medical Analysis
- **Symptom Analyzer**: Chat-based interface to analyze health concerns (Headache, Fever, Cold, etc.).
- **Report Analysis**: Capability to process medical reports.
- **Professional Guidance**: Always reminds users to seek professional medical advice.

## 🛠️ Technology Stack
- **Frontend**: HTML5, CSS3 (Glassmorphism, Modern Animations), JavaScript
- **Backend**: Node.js, Express
- **Database**: SQLite (Local)
- **Deployment**: Vercel

## ⚙️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/chamikagamage902-cloud/Bite-and-Burn-v6.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the server**:
   ```bash
   npm start
   ```
